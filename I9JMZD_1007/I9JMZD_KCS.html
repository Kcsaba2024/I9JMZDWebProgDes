SZÓKIRAKÓ JÁTÉK

Adott egy 4*9 es négyzetháló. A program megad egy - a négyzethéló feletti képhez
kapcsolódó - szót vagy mondatot, majd ennek betűit összekeverve elhelyezi ebben a
négyzethálóban.

A játékos feladata az, hogy az összekevert betűkből kirakja a szót

   vagy mondatot a kijelölt helyen. A betűket vízszintes vagy függőleges irányban egyenként
                                                                                                                          lehet mozgatni.

 

        Ha a betű útját ellálja egy másik betű, akkor az csak a másik betűig tolódik el.
Miután sikerült a beűket a megfelelő helyre mozgatni, a program ezt jelzi, ha viszont valamely
betűket felcseréltük, akkor azokra színkiemeléssel hívja fel a
figyelmüket.

Derékszögű háromszögben igaz Pitagorasz tétele:
a2 + b2 = c2

Általános háromszögek esetén igaz:
α + β + γ = 180°
