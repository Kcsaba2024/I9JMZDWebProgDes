Visual Studio Code alapvető beállításai

Bővítmények

Miután telepítettem a Visual Studio Code-ot az első lépésem az volt, hogy megnézzem, milyen bővítményekre lesz szükségem.
Mivel a html kezelésére alapértelmezetten alkalmas, így megnéztem, milyen egyébb bővítmények szükségesek egy weboldal készítéséhez.
Bővítményekre keresni a következő módon lehet: -fogaskerék- Extensions
a megjelent keresőablakba pedig beírjuk az adott nyelvet, amihez szeretnénk keresni bővítményt, vagy a bővítmény konkrét nevét. Én például a GitLens, Live Server, HTML CSS SUPPORT és a C nyelv-hez tartozó bővítményeket telepítettem.

Autosave

Az autosave beállítását a következő módon érhetjük el:
-fogaskerék- Settings- keresőablak "autosave" a Files: autosave menüpontot ha megkeressük, akkor láthatunk egy lenyíló fület, amiben többféle opciónk is van
afterDelay - minden változás után egy kis késleltetéssel ment, alapértelemezett: 600 ms én ezt tudom ajánlani mert, ennél biztosnak érzem, hogy a mentés megtörténik
onWindowChange - VSC-n belüli ablakváltás esetén ment
onFocusChange - VSC-n belüli ablakváltás esetén és bármilyen másik külső ablakra váltás esetén is ment
Aztán például van egy olyan opció, hogy csak akkor ment automatikusan, ha nincs hiba a kódban.
